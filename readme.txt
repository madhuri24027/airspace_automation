login.feature file is the test scenario file. Testcase are written in gherkin langauge using BDD framework.


Automation suite is designed using cucumber framework.

Data.properties file is used to change browser type.


Steps to run the test script:


1. Plugin the project to eclipse using maven project

2. under test folder CucumberRunner.java file is present 

3. Right click on the file and run as junit test. All the test will be executed.

or

1. install maven apache-maven-3.6.3 in the system and set the path
2. double click on start_Execution.bat 
3. execution will start in command prompt
